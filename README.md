# To-Do App

A simple and clean To-Do List application built using React and Vite.

## Live Demo

[https://to-do-list-abhishek.netlify.app/](https://to-do-list-abhishek.netlify.app/)

## About

This app allows you to manage your daily tasks easily. You can add tasks, delete them, move them up or down, and mark them as completed. The UI is clean and responsive, making it simple to organize your day.

## Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* Move tasks up
* Move tasks down
* Simple and responsive UI
* Built using React and Vite

## Tech Stack

* React
* Vite
* JavaScript
* CSS

## How to Use (Clone and Run)

1. Clone the repository

```
git clone https://github.com/Abhishekkr206/To-Do.git
```

2. Move into the project folder

```
cd To-Do
```

3. Install dependencies

```
npm install
```

4. Start the development server

```
npm run dev
```

5. Open the app
   Visit the localhost link displayed in your terminal.
